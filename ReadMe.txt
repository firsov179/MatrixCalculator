============================================
           КАЛЬКУЛЯТОР МАТРИЦ
============================================

Решение ревлизовано для матриц 10 x 10 
заполненных ЦЕЛЫМИ числами по модулю меньше или равных 30.
Реализован класс Mаtrix для хранения и операция с матрицами.

Для нахождений не целых ответов реализован класс Fraction те рациональные числа. 
Благодаря этому нет погрешностей связанных с double.

Решение систем и поиск определителя реализовано методом Гаусса.
Алгоритм поиска определителя работает за оптимальное время и память.
